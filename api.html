Developer API

We offer a simple API for developers


To initiate new mixing send GET request to https://bitmixer.io/order.php with following parameters:

addr1 - first forward address
pr1 - percent distribution for the first address
time1 - time delay for the first address
...
...
bitcode - your bitmixer code
fee - custom fee (0.5 - 3.5000)


Example:

https://bitmixer.io/order.php?addr1=16HGUokcXuJXn9yiV6uQ4N3umAWteE2cRR&pr1=33&time1=8&addr2=1F1Afwxr2xrs3ZQpf6ifqfNMxJWZt2JupK&pr2=67&time2=16&bitcode=AcOw&fee=0.6523

You may use up to 10 forward addresses.

Response:

addr={generated incoming address}&maxamount={maxamount}&bitcode={bitcode}
or error with error description.

addr - the new generated address for this mixing. Just send your BTC (during 12 hours only, less than maxamount) and order will be processed automatically.

In order to download the Letter of Guarantee you have to accept our server's cookies in previous response and get https://bitmixer.io/lettertxt.php?addr={generated address}
